# Ferramentas — SuperGerente

## Serviços Externos

| Ferramenta | Uso | Plano | Custo/mês | Dashboard |
|------------|-----|-------|-----------|-----------|
| **Railway** | Hosting backend + frontend | Usage-based | ~$5-20 | railway.app |
| **Supabase** | Database + Auth + RLS | Free/Pro | $0-25 | supabase.com |
| **Google Gemini** | IA para chat e análise | Pay-per-use | ~$5-15 | ai.google.dev |
| **Resend** | Email transacional | Free (3k/mês) | $0-20 | resend.com |
| **Kommo CRM** | CRM (conta do cliente) | Cliente paga | $0 | kommo.com |
| **GitHub** | Repositório privado | Free | $0 | github.com |
