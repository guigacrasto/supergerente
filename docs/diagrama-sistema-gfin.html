Fluxos de Dados
// Fluxo de Pagamento Operador BPO → gfin-web → Cria conta a pagar → api-service → PostgreSQL ↓ api-service → Envia p/ aprovação → Status: awaiting_approval ↓ Dono Empresa → gfin-owner → Vê aprovações pendentes ↓ Dono → Clica "Aprovar" → api-service → Status: approved ↓ Operador → Marca como pago → Status: paid → Atualiza fluxo de caixa ———————————————————————————————— // Fluxo de Importação Operador → Upload .OFX/.CSV → gfin-web (client-side read) ↓ gfin-web → POST /api/imports/preview → ofx-parser → Retorna transações ↓ Operador → Revisa, categoriza → POST /api/imports/confirm ↓ api-service → Cria payables/receivables automaticamente → Atualiza dashboard ———————————————————————————————— // Tarefas Automáticas task-scheduler (cron diário) → Verifica: ↓ Contas vencendo em 3 dias → Cria task urgente ↓ Contas vencidas → Cria task "pagamento atrasado" ↓ Extrato não conciliado 7d → Cria task "conciliação pendente" ↓ Aprovações paradas 48h → Cria task "cobrar aprovação"
Mapa de Features

Dashboard

  • KPI cards (saldo, a receber, a pagar, vencidos)
  • Gráfico fluxo de caixa 30 dias
  • Últimas movimentações
  • Widget de tarefas urgentes

Contas a Pagar

  • CRUD com fornecedor/categoria/centro de custo
  • Status: aberto → agendado → aguardando → aprovado → pago
  • Filtros por período/status/fornecedor
  • Fluxo de aprovação pelo dono

Contas a Receber

  • CRUD com cliente/categoria
  • Recebimento parcial
  • Status: pendente → recebido/atrasado/parcial
  • Filtros por período/status/cliente

Tarefas

  • Auto-geradas + manuais
  • Agrupamento por empresa ou data
  • Prioridade: baixa/média/alta/urgente
  • Clica → navega pra resolver pendência

Importação

  • Upload OFX/CSV
  • Preview antes de confirmar
  • Match automático de categorias
  • Histórico de importações

Multi-Tenant

  • Organização (BPO) com N empresas
  • 3 perfis: admin / operador / dono
  • RLS no PostgreSQL por org_id
  • Audit log com quem/quando/o que
